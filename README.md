# Deploy Cloudflare Worker secara Otomatis

Ikuti langkah-langkah di bawah ini untuk mengedit dan mendeploy **Cloudflare Worker** Anda secara otomatis. 

### Langkah-langkah Deploy

---

### 1. **Ubah File `wrangler.toml`**

- Masuk ke [**wrangler.toml**](https://github.com/Gendarxml/Gendarxml/edit/main/wrangler.toml) dan ubah nilai `name`.  
  Contoh perubahan:  
  ```toml
  name = "gendarxmlb" # Sesuaikan nama dengan kebutuhan Anda

2. Edit worker.js
	•	Ubah logika skrip Anda di worker.js sesuai dengan kebutuhan Anda.

3. Cek Status Domain
	•	Anda dapat memeriksa status dari Worker yang telah dideploy di Cek Domain.

4. Klik Deploy
	•	Setelah selesai mengedit, klik tombol di bawah untuk melakukan deploy ke Cloudflare Workers.

5. Masukkan ID dan Token Cloudflare Anda
	•	Setelah proses deploy selesai, masukkan ID Cloudflare dan Token Cloudflare Anda pada halaman konfigurasi.

Selesai! Cloudflare Worker Anda siap digunakan dan terintegrasi.

	Catatan: Pastikan Anda memiliki akses yang diperlukan untuk mengedit file dan konfigurasi Cloudflare Worker. Jika Anda mengalami masalah, coba ulangi langkah-langkah ini atau hubungi pengelola proyek.

### Perubahan yang dilakukan:
